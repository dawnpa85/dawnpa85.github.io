Bienvenidos

Apreciado estudiante, bienvenido al OVI de Diseño de Sitios Web. En esta oportunidad se le brindará a los estudiantes los principales conceptos que son fundamentales para el modelado de datos desde el punto de vista relacional, así mismo, crear, analizar y diseñar un modelo lógico de datos, para acceder de forma eficiente a la información.

Los modelos relacionales establecen alternativas para la organización y representación de la información, las cuales son almacenadas en una base de datos. Las bases de datos tienen como función proveer la integridad, seguridad y monitoreo de los datos.

Curso: Diseño de Sitios Web - 301122
Actividad: Crear un OVI en formato WEB con HTML5 y CSS3.
Grupo: 301122_53
